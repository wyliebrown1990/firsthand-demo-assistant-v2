Quick Start

  1. Search for components in the library
  2. Select a component that matches your demo context
  3. Specify a target element selector (or use auto-detect)
  4. Click "Inject Component" to add it to the page

Keyboard Shortcuts

  • Ctrl+Shift+F - Toggle debug mode
  • Ctrl+Shift+C - Clear all components

Tips

  • Use auto-detect to find suitable target elements
  • Test selectors with the validation button
  • Components are isolated using Shadow DOM for reliability
